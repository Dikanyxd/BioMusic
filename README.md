## 📖 О проекте

**BioMusic Generator** — это система, которая превращает ваше сердцебиение в живую музыку.

Датчик пульса MAX30102 считывает фотоплетизмограмму (PPG) с вашего пальца, ESP32 вычисляет частоту сердечных сокращений (BPM) и генерирует MIDI-ноты,
которые передаются по Bluetooth на Android-устройство для синтеза звука в реальном времени.

> *"Ваше сердце — дирижёр, а музыка — ваше настроение"*


✨ Возможности

| Функция | Описание |
|---------|----------|
| 💓 **Мониторинг пульса в реальном времени** | Датчик MAX30102 с точностью до удара определяет BPM |
| 🎹 **Алгоритмическая генерация музыки** | Три режима композиции с разными гаммами и паттернами |
| 📱 **Bluetooth Classic** | Передача MIDI-команд на смартфон без задержек |
| 🎚️ **Адаптивная громкость** | Громкость зависит от интенсивности пульса |
| 🎨 **Визуальная обратная связь** | Пульсирующий интерфейс, синхронизированный с вашим сердцем |

### 🎼 Музыкальные режимы

| Режим | Гамма (MIDI) | Темп | Характер |
|-------|--------------|------|----------|
| 💼 **Work** | C4–C5 (60–72) | Медленный | Спокойная, фоновая музыка |
| 🚗 **Road** | C3–C4 (48–60) | Средний | Драйвовый, предсказуемый бас |
| 🏃 **Sport** | C5–C6 (72–84) | Быстрый | Энергичные арпеджио |

## 🛠️ Технологический стек

### Аппаратная часть

| Компонент | Назначение |
|-----------|------------|
| ESP32 (NodeMCU-32S) | Основной контроллер |
| MAX30102 | Оптический датчик пульса |
| 3.3V | Питание датчика |
| GPIO21 (SDA), GPIO22 (SCL) | Шина I2C |

### Программное обеспечение

| Компонент | Технологии |
|-----------|------------|
| **Прошивка ESP32** | C++, Arduino Framework, BluetoothSerial |
| **Алгоритмы пульса** | MAX30105 + heartRate (детекция биений, скользящее среднее) |
| **Генерация музыки** | MIDI-ноты, вероятностные паттерны |
| **Android-приложение** | Qt 6, QML, QtBluetooth, QtMultimedia |
